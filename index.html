Migrace Access Forms do webového rozhraní

Marek Pilař

Zadání práce

Bakalářská práce se zabývá migrací aplikace Microsoft Access forms do webového prostředí. Stávající Access forms aplikace má data uložená na Microsoft SQL serveru a Access forms je využíván pouze jako front end pro zadávání nových dat. Účelem této práce je analyzovat stávající řešení (datová struktura, aplikační logika, use case aplikace, řízení oprávnění atd.). Navrhnout nové řešení, které by mělo mít webové rozhraní. Pro nové řešení je potřeba pomocí analýzy vybrat technologii pro tvorbu frontendu a backendu, dále je zapotřebí vybrat, zda bude služba provozována lokálně či bude fungovat za pomocí cloudu. Dále je potřeba analyzovat bezpečnost této aplikace, vyhodnotit případná rizika a vytvořit bezpečnostní omezení.

Teoretická část

  • Představení Microsoft Access
  • Představení Microsoft Access forms

  • Zhodnocení nového řešení a porovnání mezi původním a novým řešením

Praktická část

  • Aplikování oprávnění podle stanové matice oprávnění
